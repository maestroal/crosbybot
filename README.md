# crosbybot
Simple What'sApp BOT with NodeJS

## Install
+ Linux/Ubuntu
```bash
sudo apt install nodejs
sudo apt install git
sudo git clone https://github.com/maestroal/crosbybot
cd crosbybot
sudo npm i qrcode-termina
sudo npm i urlencode
sudo npm i cheerio
sudo npm i axios
sudo npm i fs
sudo npm i whatsapp-web.js

```
## Run
```bash
sudo node index
```
